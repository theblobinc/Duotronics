## Lab import rule

The lab archive contributes schemas, code-derived object boundaries, benchmark/gate patterns, adapter evidence, manifests, red-team fixtures, and migration lineage. It does not become proof of physics or lower-layer semantic authority. Lab-derived evidence is accepted only through registry, migration, replay, policy, and conformance review.
