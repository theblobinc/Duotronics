## New geometry boundary

The pronic-centered polygon profile contributes a visual-axiom geometry generator:

```text
one center dot -> even outer vertex counts -> pronic shell boundaries -> polygon chain selection -> canonical witness
```

It does not redefine DPFC arithmetic, prove chemistry/physics, or make quaternions part of the DPFC core.

The core separation is:

```text
Quaternions rotate or normalize geometry.
Pronic shells generate the polygon family.
DPFC bridges and serializes canonical objects.
The Witness Contract decides trust.
```
