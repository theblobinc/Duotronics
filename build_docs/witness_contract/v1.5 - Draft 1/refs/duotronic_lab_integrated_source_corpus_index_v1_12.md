High-speed lookup memory can support internal reasoning. Diverse models can be adjudicated as evidence sources. Planners can propose actions as candidate witnesses. Actions execute only after policy gates and produce outcome witnesses.

## v1.5 scope statement

v1.5 extends the v1.4 Draft 5 corpus into a distributed self-governing recurrent network. Nodes auto-federate through DBP v2 S2 full-duplex streams, publish computational capacity as canonical resource witnesses, receive policy-gated task delegation, and report task outcomes as witnesses. Raw metrics, Open/S1 DBP frames, stale self-models, and unadmitted nodes have zero scheduling authority.
